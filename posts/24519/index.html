Mysql修改密码

前言

  • Mysql使用频率不是很高，密码经常忘记，看了很多博客，方法很多，这里提供一个我认为最方便的
  • (1条消息) MySQL安装过程的一些问题，MySQL5.5版本忘记密码问题等_QzzBL-CSDN博客

重置方法

1.在进程中关闭mysql，或者以管理员运行cmd，输入net stop mysql

2.以管理员运行cmd,切换到Mysql安装目录

输入mysqld --skip-grant-tables，这句话表示跳过验证

3.重新开一个cmd，不用以管理员运行。同理先进入到Mysql目录下

输入 mysql -u root 表示进入mysql中

4.接着先刷新权限flush privileges;,再设置密码，引号里是你要设置的密码

测试
